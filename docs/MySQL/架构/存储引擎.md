# 存储引擎

### InnoDB

> InnoDB 是 MySQL 默认的事务型存储引擎。

InnoDB 采用 MVCC 来支持高并发，并且实现了四个标准隔离级别
- 未提交读
- 提交读
- 可重复读
- 可串行化

其默认级别是「可重复读」，在可重复读级别下，通过 MVCC + Next-Key Locking 防止幻读。

主索引是「聚簇索引」，在索引中保存了数据，从而避免直接读取磁盘，因此对主键查询有很高的性能。

### InnoDB 和 MyISAM 比较

|  | InnoDB | MyISAN |
| --- | --- | --- |
| 事务 | 支持 | 不支持 |
